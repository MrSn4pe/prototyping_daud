📱 Mockup Sistem DS UMKM

27 Desain Mobile Lengkap

  1. Landing Page 1 & CTA (Intro)

    • User scroll ke kiri-kanan untuk melihat fitur
    • User tekan tombol “Coba Sekarang”
    • User klik menu “Tentang Kami” untuk info aplikasi

  2. Landing Page 2

    • User scroll ke kiri-kanan untuk melihat fitur
    • User tekan tombol “Coba Sekarang”
    • User klik menu “Tentang Kami” untuk info aplikasi

  3. Login

    • User ketik email dan password
    • User klik tombol “Masuk”
    • User tekan link “Lupa Password” untuk reset
    • User tekan link “Daftar” untuk mendaftar

  4. Daftar

    • User isi nama, email, dan password
    • User centang “Saya menyetujui syarat & ketentuan”
    • User tekan tombol “Registrasi”

  5. Reset Password

    • User ketik email terdaftar
    • Klik tombol “Lanjutkan”
    • User tekan tombol kembali ke halaman login

  6. Verifikasi Email

    • User memasukkan kode verifikasi
    • User tekan tombol “Kirim Lagi” jika kode tidak terkirim
    • User tekan tombol “Verifikasi”

  7. Dashboard

    • User tap kartu ringkasan Grafik Penjualan, Total Pesanan, Total Stok
    • User buka menu “Pesanan Baru”
    • User klik ikon Order, Stock, Home, Laporan, Pengaturan

  8. Notifikasi

    • User scroll daftar pemberitahuan
    • User tekan notifikasi untuk menuju page tujuan
    • User tekan tombol Kembali untuk kembali

  9. Order List

    • User tekan filter “Semua, Menunggu, Dalam Proses, Selesai”
    • User tekan order untuk melihat detail
    • User scroll atas/bawah untuk melihat order lain

  10. Order Detail

    • User tekan tombol “Proses”
    • User tekan tombol “Kembali”
    • User tekan nomor telepon untuk menghubungi pelanggan via WhatsApp

  11. Konfirmasi Order

    • User tekan tombol “Ya” untuk konfirmasi
    • User tekan tombol “Tidak” untuk batal
    • User tekan luar halaman konfirmasi untuk membatalkan

  12. Stock

    • User tekan tombol “Tambah Produk”
    • User tekan tombol “Search”
    • User tekan tombol “Edit” atau “Hapus”

  13. Tambah Stock

    • User isi form Nama Produk
    • User isi form Harga
    • User tekan tombol “Tambahkan”

  14. Edit Stock

    • User pilih produk yang ingin diedit
    • User ubah jumlah stok
    • User tekan tombol “Simpan”

  15. Search Stock

    • User ketik nama produk
    • User tekan Enter untuk mencari
    • User klik luar kolom pencarian untuk menutup

  16. Hapus Produk

    • User tekan “Ya” untuk konfirmasi
    • User tekan “Tidak” untuk batal
    • User tekan luar konfirmasi untuk menutup

  17. Laporan

    • User tekan tombol “Select Date”
    • User tekan Grafik untuk detail
    • User tekan “Export Laporan”

  18. Pilihan Laporan

    • User tekan simbol pensil untuk input tanggal manual
    • User tekan tanggal untuk memilih
    • User tekan “OK” atau “Cancel”

  19. Export Laporan

    • User pilih format export
    • User pilih waktu laporan
    • User tekan tombol “Export”

  20. Laporan Detail

    • User scroll halaman laporan
    • User tekan tombol “Kembali”
    • User tekan laporan untuk melihat detail

  21. Pengaturan

    • User tekan tombol “Kembali”
    • User tekan tombol “Akun, Sistem, Notifikasi, Keamanan, Bantuan”
    • User tekan tombol “Keluar”

  22. Akun

    • User tekan “Nama” untuk ubah nama akun
    • User tekan “Email” untuk ubah email
    • User tekan “Simpan” untuk menyimpan perubahan

  23. Pengaturan Sistem

    • User ubah toggle Tema Gelap
    • User tekan “Hubungkan” untuk API Gojek
    • User tekan “Hubungkan” untuk API Grab

  24. Notifikasi

    • User aktifkan notifikasi “Pesanan Baru Masuk”
    • User aktifkan notifikasi “Stok Menipis”
    • User aktifkan notifikasi “Laporan Harian”

  25. Keamanan

    • User tekan “Ubah Password”
    • User aktifkan OTP via Email/SMS
    • User lihat “Riwayat Login”

  26. Bantuan

    • User tekan topik FAQ untuk membaca panduan
    • User ketik pesan untuk melapor kendala
    • User tekan “Kirim via WhatsApp” untuk hubungi admin
